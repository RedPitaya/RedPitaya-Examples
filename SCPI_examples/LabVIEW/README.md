# Red Pitaya LabVIEW examples

Thanks to the Framatome ATMS, a set of updated LabVIEW examples is available in a separate repository. These examples are compatible with the latest Red Pitaya firmware and provide a more user-friendly interface for controlling the Red Pitaya board.

[red-pitaya-labview-driver](https://gitlab.com/framatome-atms-community/red-pitaya-labview-driver)
